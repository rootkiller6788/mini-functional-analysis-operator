# mini-compact-operators

Compact operators, Fredholm operators, Calkin algebra, and Fredholm index theory
in Lean 4 using MiniObjectKernel.

## Package structure

- `MiniCompactOperators/Core/` — Basic definitions
- `MiniCompactOperators/Morphisms/` — Atkinson theorem, Calkin morphism
- `MiniCompactOperators/Constructions/` — Products, quotients, Schatten classes
- `MiniCompactOperators/Properties/` — Invariants, preservation, classification
- `MiniCompactOperators/Theorems/` — Main theorems (sorry-filled)
- `MiniCompactOperators/Examples/` — Standard examples and counterexamples
- `MiniCompactOperators/Bridges/` — Connections to algebra, topology, geometry, computation

## Dependencies

- mini-object-kernel
- mini-banach-hilbert
- mini-bounded-operators
